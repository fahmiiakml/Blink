Blink

Blink adalah aplikasi sederhana untuk membantu pengguna memahami kebiasaan penggunaan layar (screen time) sehari-hari. Aplikasi ini mencatat aktivitas pengguna, menampilkan ringkasan penggunaan, serta membandingkan data pengguna dengan rata-rata kelompok usia 18–24 tahun.

Tentang Aplikasi

Blink dirancang sebagai teman kecil untuk membantu pengguna lebih sadar terhadap waktu yang dihabiskan di depan layar. Aplikasi ini tidak hanya mencatat data, tetapi juga memberikan gambaran sederhana tentang bagaimana kebiasaan pengguna dibandingkan dengan orang lain dalam kelompok usia yang sama.

Fitur Utama
Mencatat screen time harian
Menyimpan data sleep, stress, dan mood
Menampilkan riwayat input pengguna
Perbandingan dengan rata-rata pengguna usia 18–24
Statistik sederhana untuk melihat posisi pengguna
Halaman informasi dan bantuan
